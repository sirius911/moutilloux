## Problèmes d'orchestration

### 🟡 index.lock résiduel après chaque opération git (sandbox → FUSE macOS)

Le sandbox ne peut pas supprimer les fichiers temporaires créés par git dans `.git/`
(`unlink: Operation not permitted`). Chaque opération git réussit mais laisse un
lock orphelin. Contournement utilisé dans cette session :
- `git symbolic-ref HEAD ...` pour changer de branche sans lock
- `git commit-tree` + écriture directe du ref pour corriger le parent du commit
- Push à faire manuellement depuis la machine macOS (credentials HTTPS absents du sandbox)

**Action humaine requise :**
```bash
git push origin sprint/02-admin-tournoi
```
