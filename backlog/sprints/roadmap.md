# Roadmap — Sprints

> 1. Supprimer sa ligne de cette liste
> 2. Déplacer le dossier `NN-nom/` dans `done/`
> 3. Le sprint suivant dans la liste devient automatiquement le courant
>
> Si cette liste est vide → le SESSION_ENGINE désactive la tâche planifiée.
